jQuery核心函数

​ 1. jQuery(callback) ←当DOM加载完成后要执行的函数

​ $(函数) ←页面加载运行的前提

​ 2. jQuery(html,*[ownerDocument]*) ←根据 HTML 代码表示的元素标签，转化成jQuery对象

​ 3. jQuery([selector,[context]]) ←根据选择器,把dom对象转化成jQuery对象

Dom对象转换成jQuery对象,jQuery对象转换成Dom对象
Dom对象转换成jQuery对象: $(dom对象)
jQuery对象转换成Dom对象: jquery对象[0] 或者 jquery对象.get(0)

核心目的:把页面的标签转换成jQuery对象,然后调用jQuery中提供的相关函数和属性进行操作该标签

常用函数总结:

​ html ()方法，无参数代表获取元素中的html代码，如果有参数代表设置元素的html信息

​ text()方法，无参数代表获取元素中的文本信息，如果有参数代表设置元素的文本信息

​ each()方法，它的使用可以帮助我们遍历元素，可以传递一个函数 each(function(n)), n就代表的是每一个元素的序号，在函数中可以通过this来获取其dom对象index()方法，它可以判断元素的序号

​ css()方法，它可以设置元素的样式,如果只有一个值 css(name,value),如果有多个值,css({name:value,name:value})

​ attr()设置或返回被选元素的属性值

​ val() 获取元素当前的值 value值show() 让元素显示出来hide() 让元素隐藏起来

​ trim()它是去掉字符串左右空格

jQuery选择器

基本选择器

#id 根据 id属性查找一个元素

Element 根据元素的标签名称查找元素

.class 根据元素的class属性查找元素

*匹配所有元素

Select1,selector2…将每一个选择器匹配到元素合并后一起返回

层级选择器

  1. 空格 得到元素下的所有子元素(包含直接子元素与间接子元素)
  2. 大于号> 只获取直接的子元素
  3. 加号+ 只获取直接的一个兄弟元素
  4. 波浪线~ 所有的后边的兄弟元素
